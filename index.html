Desktop · OCR · Géomatique 2026

ArchiScan — Numérisation et indexation d'archives

Application desktop pour numériser des documents anciens, extraire automatiquement les entités (noms, dates, lieux) via OCR et NLP, puis les géolocaliser sur une carte interactive PostGIS.

C# WinForm PostgreSQL/PostGIS NHibernate GDAL Tesseract OCR spaCy Merise GitLab

Contexte : les archives papier (lettres familiales, certificats, photos anciennes) accumulent une richesse historique difficile à exploiter. ArchiScan automatise la chaîne complète : numérisation → extraction de texte → identification d'entités → cartographie interactive.

Architecture (3 couches) : frontend WinForm C# · couche métier avec NHibernate · backend PostgreSQL/PostGIS. Modules transverses : OCR/NLP (Tesseract + spaCy) et géospatial (GDAL.NET).

Fonctionnalités : numérisation par lots · OCR multilingue avec correction manuelle · extraction d'entités nommées (NER) · géocodage automatique des lieux · visualisation cartographique interactive · recherche full-text avec filtres temporels et géographiques.

Modèle de données : schéma Merise avec entités Document, Personne, Lieu, Événement, Tag — relations N:N.

Statut : projet personnel en cours, conçu pour explorer la chaîne C#/.NET + PostGIS + IA appliquée au traitement de documents.

Projet en cours de développement — code source disponible sur demande.

PFE · Géomatique · Vision 2022

Image-to-Image Translation — Satellite → Cartes (PFE ENSIAS)

Projet de géomatique appliquée : deux GAN conditionnels (Pix2Pix et CycleGAN) pour traduire automatiquement des images satellites en cartes lisibles. Rapport complet publié en open source.

Pix2Pix CycleGAN U-Net ResNet TensorFlow OpenCV

Contexte : mémoire de fin d'études du Master Sciences des Données & Big Data à l'ENSIAS (Rabat, 2022).

Enjeu : comparer un modèle supervisé (Pix2Pix) et un modèle non supervisé (CycleGAN) sur le même jeu de données satellitaire.

Publication : rapport diffusé sur GitHub sous licence Creative Commons BY-NC-ND 4.0.

Web · IA · Accessibilité 2025

ReadZen — Lecteur de documents intelligent

Plateforme de lecture et de gestion de documents professionnels : indexation automatique par domaine, résumés IA, carte mentale, traduction multilingue et mode dyslexie. Projet livré en 48 h au Hackathon CAF 2025.

React · TypeScript Vite · Tailwind Node.js · Express MongoDB OpenAI Tesseract.js
